MERN stack with Redux

This project has an extensive backend API with & Express. User Authentication was easily done by using AuthO. .

React is generally fast, but by default any updates to a component will cause React to re-render all of the components inside that part of the component tree. REDUX was implemented to solve the predictable state updates, ensure that each connected component only extracts the specific pieces of data from the store state that are needed by that component.
